# Testing RedwoodJS Forms

* `FormTest` - a JavaScript example built using [RedwoodJS' form components](https://redwoodjs.com/docs/form.html).
* `OurForm` - a TypeScript example built using [react-hook-form](https://react-hook-form.com/).

A detailed explanation can be found on the [RedwoodJS forum](https://community.redwoodjs.com/t/testing-forms-using-testing-library-user-event/2058) and in the [RedwoodJS documentation](https://redwoodjs.com/docs/testing#testing-forms).
